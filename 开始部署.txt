第1步：创建GitHub仓库
   • 访问 github.com/new
   • 仓库名：reciting
   • 设置为Public（公开）
   • 不要勾选"Add a README"
   • 点击创建

第2步：上传代码
   
   方式A - 命令行（推荐）：
   
   git init
   git add .
   git commit -m "首次提交：词忆项目"
   git remote add origin https://github.com/你的用户名/reciting.git
   git branch -M main
   git push -u origin main
   
   方式B - GitHub Desktop：
   • 下载 GitHub Desktop
   • Add Local Repository
   • Publish repository

第3步：启用GitHub Pages
   • 仓库页面 → Settings → Pages
   • Branch: main
   • Folder: / (root)
   • 点击 Save
   • 等待1-2分钟

✅ 完成！访问：https://你的用户名.github.io/reciting/
