## 🚀 部署步骤（完成上述任务后）

### 第1步：创建GitHub仓库

1. 登录 [GitHub](https://github.com)
2. 点击右上角 `+` → `New repository`
3. 填写信息：
   - **Repository name**: `reciting`
   - **Description**: `词忆 - 高效背单词工具`
   - **Public** ✅ （必须是公开的）
   - ❌ 不要勾选 "Add a README file"
   - **License**: MIT License（推荐）
4. 点击 `Create repository`

✅ 完成：
- [ ] 已创建GitHub仓库

---

### 第2步：上传代码

#### 方式A：使用命令行（推荐）

在项目目录打开终端，执行：

```bash
# 1. 初始化Git仓库
git init

# 2. 添加所有文件
git add .

# 3. 创建第一次提交
git commit -m "首次提交：词忆背单词项目"

# 4. 关联远程仓库（替换你的用户名）
git remote add origin https://github.com/你的用户名/reciting.git

# 5. 推送到GitHub
git branch -M main
git push -u origin main
```

#### 方式B：使用GitHub Desktop（适合新手）

1. 下载 [GitHub Desktop](https://desktop.github.com/)
2. 登录GitHub账号
3. `File` → `Add Local Repository`
4. 选择项目文件夹
5. 点击 `Publish repository`

✅ 完成：
- [ ] 代码已上传到GitHub

---

### 第3步：启用GitHub Pages

1. 在GitHub仓库页面，点击 `Settings`
2. 左侧菜单找到 `Pages`
3. **Source** 部分设置：
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. 点击 `Save`
5. 等待1-2分钟

✅ 完成：
- [ ] GitHub Pages已启用
- [ ] 获得了访问链接

**你的网站地址**：
```
https://你的用户名.github.io/reciting/
```
