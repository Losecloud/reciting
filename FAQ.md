# 常见问题解答（FAQ）

## 📱 使用相关

### Q1: 如何导入词书？

**A:** 有三种方式：

1. **拖拽上传**：直接拖拽文件到上传区域
2. **点击上传**：点击"添加词书"按钮，选择文件
3. **示例词书**：使用 `examples/` 目录下的示例文件

支持的格式：CSV、TXT、Excel(.xlsx)

### Q2: 词书格式要求是什么？

**A:** 

**CSV格式**（推荐）：
```csv
单词,音标,词性,释义,例句,词根词缀,例句翻译
word,[wɜːd],n.,单词,This is a word.,word-,这是一个单词。
```

**TXT格式**（简单）：
```
word 单词
study 学习
```

至少需要包含"单词"和"释义"两列。

### Q3: 数据保存在哪里？

**A:** 所有数据保存在浏览器的 `localStorage` 中，包括：
- 词书内容
- 学习进度
- 复习计划
- 个人设置

数据只存在你的设备上，不会上传到任何服务器。

### Q4: 数据会丢失吗？

**A:** 在以下情况可能丢失：
- ❌ 清除浏览器数据/缓存
- ❌ 使用隐私模式/无痕模式
- ❌ 卸载浏览器
- ❌ 重装系统

**建议**：
- ✅ 定期导出数据备份
- ✅ 不要清除浏览器数据
- ✅ 使用正常浏览模式

### Q5: 如何备份数据？

**A:** 
1. 打开词书设置
2. 点击"导出词书"按钮
3. 保存CSV文件
4. 下次可以重新导入

**注意**：目前无法导出学习进度，只能导出词书内容。

### Q6: 可以在多个设备间同步吗？

**A:** 目前不支持自动同步。每个设备的数据是独立的。

**变通方法**：
- 通过导出/导入手动同步词书
- 等待未来的云同步功能（计划中）

## 📖 学习相关

### Q7: 复习时间是如何计算的？

**A:** 基于艾宾浩斯遗忘曲线：
- 第1次复习：5分钟后
- 第2次复习：30分钟后
- 第3次复习：12小时后
- 第4次复习：1天后
- 第5次复习：2天后
- 第6次复习：4天后
- 第7次复习：7天后
- 第8次复习：15天后

答错的单词会重新计算。

### Q8: 如何修改复习间隔？

**A:** 
1. 点击右上角"设置"按钮
2. 找到"复习间隔"设置
3. 自定义各阶段的时间
4. 保存设置

### Q9: "认识"、"不认识"、"模糊"有什么区别？

**A:** 
- **认识**：单词进入复习队列，按计划复习
- **不认识**：标记为生词，继续在本轮学习
- **模糊**：类似"认识"，但复习间隔稍短

### Q10: 可以跳过已会的单词吗？

**A:** 可以！在学习时选择"认识"，系统会安排复习。如果连续多次答对，复习间隔会越来越长。

## 📱 移动端相关

### Q11: 如何添加到手机桌面？

**A:** 

**iOS (Safari)**：
1. 打开网站
2. 点击底部 📤 分享按钮
3. 选择"添加到主屏幕"
4. 点击"添加"

**Android (Chrome)**：
1. 打开网站
2. 点击右上角 ⋮ 菜单
3. 选择"添加到主屏幕"或"安装应用"
4. 点击"安装"

### Q12: 添加到桌面后有什么好处？

**A:** 
- ✅ 像APP一样启动，全屏显示
- ✅ 支持离线使用
- ✅ 加载更快
- ✅ 独立的窗口，不受浏览器干扰

### Q13: 可以离线使用吗？

**A:** 可以！添加到桌面后，即使断网也能使用。

**注意**：
- 首次访问需要联网
- 离线状态下无法使用需要网络的功能（如AI工坊）

### Q15: 为什么有些功能不可用？

**A:** 可能原因：
- 浏览器版本太旧
- 浏览器不支持某些API
- 使用隐私模式

**解决方法**：
- 更新浏览器到最新版本
- 使用推荐的浏览器
- 不要使用隐私/无痕模式

### Q16: Service Worker是什么？

**A:** Service Worker 是一种后台脚本，用于：
- 缓存资源，实现离线访问
- 提升加载速度
- 支持PWA功能

不会收集或上传任何数据。

### Q17: 为什么需要图标文件？

**A:** 图标用于：
- 添加到桌面时显示
- PWA应用图标
- 浏览器标签页图标

如果没有图标，添加到桌面时会显示默认图标。
